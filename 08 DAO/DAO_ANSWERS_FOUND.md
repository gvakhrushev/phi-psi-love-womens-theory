## 1. ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ

### Определение последней цифры
**Из 8.1_phi_dao_spec.txt, строки 107-116:**
```solidity
function _extractDoorFromAmount(uint256 amount) internal pure returns (uint8) {
    // Берем последнюю значащую цифру
    uint256 normalized = amount / 10000; // Убрать лишние нули
    return uint8(normalized % 10);
}
```
- Сумма нормализуется делением на 10000
- Берётся остаток от деления на 10
- Примеры из файла (строки 5-17):
  - 0.01 = команда 1
  - 0.001 = команда 1
  - 0.02 = команда 2
  - 0.00222 = команда 2
  - 0.00 = команда 0

## 3. СИСТЕМА РОЛЕЙ 38/62

### Что означают роли
**Из 8.3_queue_system_design.md, строки 9-18:**
```
РОЛЬ-62 "СОХРАНЕНИЕ" (Хранитель)
├─ Получает: 62% от распределений
├─ Функция: накопление, долгосрочная память, поддержание
└─ Метафора: сохранение огня цивилизации

РОЛЬ-38 "ЗАЩИТА" (Защитник)
├─ Получает: 38% от распределений
├─ Функция: расширение, добыча, оборона периметра
└─ Метафора: защита границ системы
```

### Почему "негендерные"
**Из 8.3_queue_system_design.md, строка 4:**
"Каждый человек сам выбирает свою роль в экосистеме, независимо от гендера"
- Роли не привязаны к полу
- Свободный выбор при регистрации
- Можно менять роль в любой момент (для будущих выплат)

## 4. СИСТЕМА ОЧЕРЕДЕЙ

### Приоритет новичкам + возраст
**Из 8.3_queue_system_design.md, строки 48-54:**
```
ПРИНЦИП: Новички (дети) имеют приоритет, но справедливость через возраст

Очередь 1: Первая выплата (для новичков)
Очередь 2: Вторая выплата (минимальный возраст)
Очередь 3: Третья выплата (больший возраст)
```

**Алгоритм (строки 68-82):**
- Новички входят в очередь 1
- С возрастом кошелька открываются дополнительные очереди
- Возраст = время с момента создания кошелька в блокчейне

## 5. ЗАЩИТА ОТ АТАК

### Антихрупкость и 38.2% порог
**Из 8.1_phi_dao_spec.txt, строка 34:**
```solidity
uint256 constant PHI2_THRESHOLD = 382;  // 38.2% для одобрения
```

**Из README, строка 78:**
"38.2% порог — природный оптимум консенсуса (φ²)"

**Из README, строка 79:**
"Свободный выход — защита от захвата власти"
- Можно вывести liquid часть (61.8%) в любой момент
