## B1. Римановская гипотеза (RH)
**Классика:** все нетривиальные нули \(\zeta(s)\) имеют \(\operatorname{Re}s=\tfrac12\).

### B1.1. Встраивание в T_all (φ–ψ CoP)
- **Сцена:** спектр \(\zeta\) читается как **код различений** на φ‑решётке; зеркальность реализуется **внутренним** каналом ψ (переворот/обратимость), а не внешним ярлыком.
- **Баланс:** \(\operatorname{Re}s=\tfrac12\) — точка **ψ‑инвариантности** (φ↔ψ) с минимальной ценой описания.

### B1.2. Инвариант и min‑K
- Берём симметризованный функционал \(\Xi(s)\) и определяем **инвариант баланса**
  \[ I(s) := \Bigl| \log \frac{\Xi(s)}{\Xi(1-\overline{s})} \Bigr|. \]
  При **ψ‑инвариантности** \(I(s)=0\). Нули на \(\operatorname{Re}s=\tfrac12\) дают **min‑K** код спектра (нет ярлыка «относительно чего симметрия»).

### B1.3. Редукция (скетч)
1) В T_all зеркальность — **внутренняя** (ψ), внешние флаги запрещены.
2) Нуль с \(\operatorname{Re}s\neq \tfrac12\) требует внешней метки, чтобы сохранить зеркальность ⇒ \(\Delta K>0\).
3) Альтернатива, не уменьшающая K при TR=1, — эпицикл.

### B1.4. Фальсификация
- Показать устойчивые нули \(\operatorname{Re}s\neq \tfrac12\) с \(I(s)=0\) **без** внешних ярлыков; либо предъявить более короткий инвариант, совместимый с TR=1.

### B1.5. Мини‑проверки
- По данным Одлыжко: локальные ψ‑вариации **не** сдвигают нули с \(\tfrac12\) (\(I\) остаётся нулём).

### B1.6. Псевдокод проверки \(I(s)\)
```
Input: список нулей {ρ_k} с высокой точностью; процедура оценки Xi(s)
For each ρ_k near Re=1/2:
  ε ← малый сдвиг ориентации (ψ-вариация) в ядре функционала
  I0 ← |log( Xi(ρ_k) / Xi(1 - \bar{ρ_k}) )|
  Iε ← |log( Xi(ρ_k+ε) / Xi(1 - \overline{(ρ_k+ε)}) )|
  assert I0 ≈ 0 and Iε ≈ 0  // ψ-устойчивость
fail if exists k: I0>tol or Iε>tol
```

### B1.7. Границы фальсификации
Если находится бесконечная подпоследовательность нулей с \(\operatorname{Re}s\ne 1/2\), для которой \(I\equiv 0\) при ψ‑вариациях **и** длина описания не возрастает, CoP‑редукция RH опровергнута.

## B2. Берч—Свиннертон‑Даер (BSD)
**Классика:** ранг \(E(\mathbb Q)\) равен порядку нуля \(L(E,s)\) при \(s=1\).

### B2.1. Встраивание в T_all
- Независимые генераторы \(E(\mathbb Q)\) читаются как **ψ‑циклы** на φ‑решётке. Ориентация/инверсия цикла — внутренняя ψ.

### B2.2. Инвариант и min‑K
- **Инвариант ранга:**
  \[ R_{\psi}(E) := \operatorname{ord}_{s=1} L(E,s). \]
  Это **min‑K** счётчик ψ‑циклов: внешние «регуляторы/корректировки» ранга — ярлыки ⇒ \(\Delta K>0\).

### B2.3. Редукция (скетч)
1) Ранг — число ψ‑циклов **в ядре**.
2) \(\operatorname{ord}_{s=1} L\) — единственный короткий счётчик тех же циклов при TR=1.
3) Любой альтернативный счётчик, требующий ярлыков, нарушает стоп‑правило.

### B2.4. Фальсификация
- Найти семейство кривых, где равенство нарушено **без** внешних ярлыков и при сохранении TR=1; либо предъявить ещё более короткий ψ‑счётчик.

### B2.5. Мини‑проверки
- Малые проводимости: устойчивость \(R_{\psi}(E)\) под изогениями (ψ‑вариации) без роста K.

### B2.6. Лемма (ψ‑инвариантность под изогениями)
Если \(E\) и \(E'\) изогонны над \(\mathbb Q\), то при ψ‑встраивании ядра выполняется
\[ R_{\psi}(E) = R_{\psi}(E') = \operatorname{ord}_{s=1} L(E,s). \]
Любое расхождение потребует внешнего ярлыка для переопределения базиса → \(\Delta K>0\).

### B2.7. Псевдокод сопоставления
```
Input: семейство кривых {E_i} малой проводимости; таблица L(E_i,s)
For each E_i:
  r_i ← ord_{s=1} L(E_i,s)
  for изогению φ: E_i → E_j:
    assert r_i == ord_{s=1} L(E_j,s)
  // ψ-вариация базиса не меняет r
fail if существует пара (i,j): r_i ≠ r_j при изогении
```

## B3. Гипотеза Ходжа
**Классика:** каждый рациональный (p,p)-класс когомологии — класс алгебраического цикла.

### B3.1. Встраивание в T_all
- (p,p)-классы = **ψ‑голономии** (память пути) на φ‑решётке; «алгебраичность» — внутреннее свойство **min‑K**, не внешний тег.

### B3.2. Инвариант и min‑K
- **Функционал выбора:**
  \[ \mathcal{H}(\omega) := \min\{\mathrm{DL}(\text{цикла}),\ K(\text{класса})\}\ \text{при ψ‑замкнутости}. \]
  Минимум достигается на алгебраических циклах **без** внешних меток.

### B3.3. Редукция (скетч)
1) Требуем ψ‑замкнутость: \(r\mapsto 1/r\), \(r\mapsto -r\) не добавляют ярлыков.
2) Если «алгебраичность» держится только с меткой — это \(\Delta K>0\) ⇒ не ядро.
3) Следовательно, алгебраические циклы — **минимальные носители** памяти (ψ) и дают минимум \(\mathcal{H}\).

### B3.4. Фальсификация
- Предъявить (p,p)-класс, для которого **ни один** алгебраический представитель не достигает min‑K без внешних ярлыков при TR=1.

### B3.5. Мини‑проверки
- K3/абелевы: явная ψ‑реализация стандартных (1,1)-классов и сравнение DL/K с неалгебраическими представлениями.

### B3.6. Критерий ψ‑замкнутости (операционный)
Класс \([\omega]\in H^{p,p}\) ψ‑замкнут, если существует сеть циклов \(\{C_i\}\) и форм \(\{\eta_i\}\) такая, что для всякой петли \(\gamma\)
\[ \sum_i \int_{C_i\circ\gamma} \eta_i = 0, \]
и замена \(\gamma\mapsto\gamma^{-1}\) не меняет описательную длину. Любая реализация, где требуется ярлык направления, нарушает min‑K.

### B3.7. Пример: K3
Для K3‑поверхности стандартные (1,1)-классы реализуют ψ‑замкнутость без внешних меток; сравнение с трансцендентными классами показывает рост DL/K при попытке сохранить ψ‑инвариантность.

## B4. Масс‑gap Янга—Миллса (YM)
**Классика:** квантовая SU(N) YM на \(\mathbb R^4\) существует и имеет положительный масс‑gap.

### B4.1. Встраивание в T_all
- Вакуум YM — сеть топосекторов; **ψ‑запирание** (ориентация цикла внутри) порождает **энергетический порог**.

### B4.2. Инвариант и min‑K
- **Нижняя оценка:**
  \[ m_{\mathrm{gap}} \ge c_\phi\,\Lambda, \qquad c_\phi \sim \phi, \]
  где \(\Lambda\) — естественный масштаб, а коэффициент — **ψ‑инвариант** (баланс φ↔ψ). Любые «сечения» вне ядра ⇒ ярлыки ⇒ \(\Delta K>0\).

### B4.3. Редукция (скетч)
1) Конфинемент = внутреннее ψ‑запирание ⇒ min‑K.
2) Отсутствие gap требует внешнего флага для обратимости/ориентации ⇒ рост K.
3) Следовательно, min‑K реализуется на конфайнменте с положительным порогом.

### B4.4. Фальсификация
- Построить согласованную квантовую SU(N) YM **без** gap при TR=1 и **без** роста K (без ярлыков/сечений).

### B4.5. Мини‑проверки
- Решётка: стабильность порога при ψ‑вариациях калибровки; CoP‑форма короткой оценки снизу (неравенственная лемма).

### B4.6. Неравенственная лемма (каркас)
Существует константа \(c_\phi>0\), зависящая лишь от ψ‑инварианта, такая что для любого конфигурационного сектора
\[ \langle H \rangle \ge c_\phi\,\Lambda, \]
и равенство невозможно без вынесения ψ наружу. Любая процедура, нарушающая оценку, обязана вводить внешний ярлык ⇒ \(\Delta K>0\).

### B4.7. Протокол решёточной проверки
```
1) Выбрать калибровки, отличающиеся ψ-вариациями, при фиксированном объёме.
2) Оценить m_gap по стандартным корреляторам.
3) Проверить инвариантность порога относительно ψ-вариаций при неизменном K.
fail, если найден режим без gap без роста описания.
```

## B5. Гладкость/существование Навье—Стокса (NS)
**Классика:** в \(\mathbb R^3\) для гладких данных — глобальная гладкость/или взрыв.

### B5.1. Встраивание в T_all
- Энерго‑каскад — перенос «различимости» по масштабам; **ψ‑голономия** удерживает обратимость и запрещает «адресные» ярлыки ⇒ подавление blow‑up.

### B5.2. Инвариант и min‑K
- **Барьер гладкости (идея):** существует функционал \(\mathcal{E}_\psi[u]\) (ψ‑голономия по вихрю/энергии), такой что
  \[ \forall t\ge0:\ \mathcal{E}_\psi[u(t)] \le \mathcal{E}_\psi[u(0)] < C_\phi \Rightarrow u\in C^\infty(\mathbb R^3\times[0,\infty)). \]
  Любой blow‑up требует внешней метки ⇒ \(\Delta K>0\).

### B5.3. Редукция (скетч)
1) Гладкость = удержание ψ внутри при min‑K на всём времени.
2) Blow‑up = появление внешнего ярлыка (адресность переноса) ⇒ рост K ⇒ не минимально.

### B5.4. Фальсификация
- Построить blow‑up **без** внешних ярлыков и со **снижением** K; либо показать невозможность определения ψ‑барьера при TR=1.

### B5.5. Мини‑проверки
- Тест Тейлора–Грина: внедрить ψ‑голономию, проверить численно монотонность \(\mathcal{E}_\psi\); аналитически — короткая априорная оценка в согласованной норме.

### B5.6. Кандидат функционала \(\mathcal{E}_\psi\)
\[ \mathcal{E}_\psi[u] := \int_{\mathbb R^3} \Big( |u|^2 + \lambda_\phi\,|\omega|^2 + \mu_\psi\,|\nabla\times(\Pi_\psi u)|^2 \Big)\,dx, \]
где \(\omega=\nabla\times u\), а \(\Pi_\psi\) — проектор на ψ‑ориентированный подпространственный поток. Параметры \(\lambda_\phi,\mu_\psi\) фиксируются из условия min‑K (без внешних ярлыков).

### B5.7. Численный протокол
```
1) Инициализировать данные Тейлор–Грина на решётке N^3.
2) Включить оператор Π_ψ (локальная инверсия/голономия) без внешних ярлыков.
3) Эволюционировать u(t) и мониторить E_ψ[u(t)].
4) Проверять монотонность/барьер при вариациях ψ, не меняя K-схемы.
fail, если blow-up происходит при неизменном или падающем K без ψ-ярлыков.
```

## II. Минимальные решения B1–B5 (королларии φ–ψ CoP)

### B1. Гипотеза Римана (RH) — минимальное решение
**Утверждение (min).** Критическая прямая `Re(s)=1/2` — единственная конфигурация, позволяющая закодировать **всё множество нулей** ζ **единым генератором** без внешних ярлыков при **min‑K** и **ψ‑симметриях** внутри (TR=1). Любой off‑сдвиг требует дополнительного кода ⇒ **ΔK>0**.  
**Шаблон редукции.** Носитель: φ‑решётка; инварианты: ψ‑флип, min‑K на бесконечном множестве; критерий: тотальность генератора.  
**Окно фальсификации.** Дать **единый внутренний генератор** off‑нулей с `K ≤ K(1/2)` при TR=1.

### B2. Birch–Swinnerton‑Dyer (BSD) — минимальное решение
**Утверждение (min).** `ord_{s=1}L(E,s) = rank E(ℚ)` — тождество **двух внутренних кодов** одной сцены при **min‑K** и ψ‑согласовании (TR=1). Несоответствие потребовало бы внешний ярлык ⇒ **ΔK>0**.  
**Шаблон редукции.** Спектр ↔ морфизм кодируют один класс без внешних адресовок.  
**Окно фальсификации.** Семейство с устойчивым рассогласованием **без** ↑K при TR=1.

### B3. Hodge — минимальное решение
**Утверждение (min).** Гармонический представитель на алгебраических циклах минимизирует описание. «Неалгебраический минимум» требует внешнего ярлыка ⇒ **ΔK>0**.  
**Шаблон редукции.** Гармоника = внутренний ψ‑баланс; алгебраичность = отсутствие ярлыков.  
**Окно фальсификации.** Класс, где неалгебраическая гармоника даёт **K↓** при TR=1.

### B4. Yang–Mills (mass gap) — минимальное решение
**Утверждение (min).** Положительный gap — ценник поддержания ψ‑голономии на горизонте при **нулевом зазоре** сцены и `γ=2`; сползание m→0 потребовало бы внешний масштаб ⇒ **ΔK>0**.  
**Шаблон редукции.** Масштаб ↔ память на «коже».  
**Окно фальсификации.** Конфигурации m→0 с конечной энергией **без** ↑K при TR=1.

### B5. Navier–Stokes — минимальное решение
**Утверждение (min).** Гладкость/энергетическая ограниченность совместимы с **min‑K** и **ψ‑внутри**; допустимый взрыв потребует внешней адресации ⇒ **ΔK>0**.  
**Шаблон редукции.** Два потока (геометрия/динамика) с внутренней памятью.  
**Окно фальсификации.** Взрыв при TR=1 **без** внешних ярлыков и **без** ↑K.

## V. Заключение

Пять задач сведены к одному принципу: **минимальная сцена без внешних ярлыков**. Это короче, строже и шире. Любая «альтернатива» с оракулами или подменой K — вне протокола (эпицикл).
