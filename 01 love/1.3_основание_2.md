### 1.1. φ-CoP карточка: США рабство

**SCENE:**
```
Территория: Северная Америка, колонии/США
Период: 1619-1865 (246 лет)
Институты: законы, суды, полиция, экономика
Сужение: "белые > черные" (расовая иерархия)
```

**QUESTION:** Как долго может существовать институциональное сужение?

**INVARIANTS:**
```
Сужение: рабство (полная собственность)
Институты: 
- Fugitive Slave Act (1850): обязанность возврата беглых
- Dred Scott v. Sandford (1857): рабы ≠ граждане
- Конституция: рабы = 3/5 человека для переписи
Измеримо:
- Ввезено: ~388,000 африканцев (1619-1808)
- Родилось в рабстве: ~4 млн (к 1860)
- Смертность: высокая (нет точных данных, тысячи ежегодно)
```

**VERIFICATION:**
- Исторические документы: переписи населения, законы
- Экономические данные: стоимость рабов, производство хлопка
- Объективно: до 1865 рабство = легально

```
Место: Версаль, Франция
Контекст: окончание WWI
```

**QUESTION:** Что создаёт Гитлеров?

**CONDITION (невыполнимое):**
```

- Репарации: 132 млрд золотых марок (~$442 млрд)
- ВВП Германии 1919: ~$12 млрд/год
- Соотношение: 132/12 = 37 лет ВВП
- Срок: до 1988 года (69 лет)
- БЕЗ: армии (max 100k), территорий (-13%), ресурсов (Рур, Саар)
```

**RESULT:**
```

  - 1918: 1 буханка = 0.63 марки
```

**INVARIANT:**

```
L(t) = ∫[t₀ to t] r(τ) dτ

где:
L = потери (жизни)
t₀ = момент сигнала (игнорирование началось)
t = текущий момент
r(τ) = скорость потерь в момент τ
```

## VI. ДИАГНОЗ

### φ-CoP формализация

**THEOREM (структурный фашизм):**
```
∀ система S с институтами I:

ЕСЛИ:
1. ∃ сужение Σ → Σ' где dim(Σ') < dim(Σ)
2. I поддерживает сужение систематически (T > T_threshold)
3. ∃ измеримые жертвы C(T) > 0
4. ∃ сигнал опровержения в момент t₀
5. Система игнорирует: t - t₀ > T_ignore

ТО:
S демонстрирует структурный_фашизм

НЕЗАВИСИМО ОТ:
- политической идеологии
- географии
- эпохи
```

**COROLLARY:**
```
Фашизм ≠ только политический
Фашизм = структурное явление

Проявления:
- Политический (Гитлер, Сталин)
- Экономический (монополии, рабство)
- Интеллектуальный (CMI, комитеты)
```

## VII. ФАЛЬСИФИКАЦИЯ

### Как опровергнуть

**Опровергнуть можно:**

1. **Показать, что счётчик неверен:**
   - Найти ошибки в исторических данных
   - Показать, что задержки НЕ приводят к жертвам
   - Предоставить альтернативные данные

2. **Показать, что паттерн ложен:**
   - Найти случай невыполнимого условия БЕЗ катастрофы
   - Доказать, что Гитлер НЕ продукт Версаля
   - Показать сигнал, который услышали немедленно

3. **Оспорить структурное определение:**
   - Показать, что рабство ≠ фашизм
   - Показать, что сегрегация ≠ фашизм
   - Объяснить различие БЕЗ ad hoc исключений

4. **Показать контрпример:**
   - Институт с невыполнимым условием БЕЗ жертв
   - Систему, которая услышала сигнал немедленно

**Если не можете → теорема верна.**

## IX. РЕЗЮМЕ (1 абзац)

Фашизм — структурное явление (систематическое сужение через институты с измеримыми жертвами), не идеология. Рабство США (1619-1865: 4+ млн) → сегрегация (1865-1968: 4,743 линчевано) = 349 лет расового фашизма. Джесси Оуэнс опроверг расизм публично (1936: 4 золота), но США И Германия проигнорировали: США продолжили сегрегацию 32 года, Германия начала войну через 3 года (75 млн смертей). Версальский договор 1919 создал невыполнимое условие → Гитлер = симптом, не причина. Перельман опроверг эпициклизм (2006: отказ от наград), CMI игнорирует 20 лет → 7.5 млн смертей от задержки парадигмы (доля CMI: 75k-375k). Система создаёт монстров через невыполнимые условия. φ-DAO иск 2025 = предотвращение F7 ("академический Гитлер" 2030-х). Счётчик: +1,027 смертей/день. Каждый день промедления = измеримые жертвы.
