# 🏗️ АРХИТЕКТУРА φ-DAO: ПОЛНАЯ СХЕМА v2.0

## 🎯 ГЛАВНАЯ КОНЦЕПЦИЯ
**φ-роутер** = Универсальный механизм распределения через двери
- **НЕ комнаты** (один вход) → **ДВЕРИ** (можно зайти и выйти)
- **Отказ от изменения** при запуске → только добавление новых дверей
- **Антихрупкость** через эволюционное развитие (как мицелий)

### Защита от атак
1. **Sybil атаки** → DID через humanity.org (биометрия рук)
2. **51% атаки** → 38.2% порог (φ²) вместо 50%
3. **Захват власти** → Свободный выход (61.8% liquid)
4. **Изменение кода** → Отказ от изменений при запуске
5. **Спам** → Автовозврат при неверной двери
