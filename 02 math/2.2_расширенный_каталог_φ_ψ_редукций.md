## РАЗДЕЛ A: ТЕОРИЯ ЧИСЕЛ

### A1. Гипотеза Гольдбаха

**Классическая формулировка:** Каждое чётное число n > 2 есть сумма двух простых чисел.

#### A1.1. Встраивание в T_all (φ-ψ CoP)

**Сцена:** Аддитивная структура простых — это **ψ-сшивка** двух простых спектров на φ-решётке. Внешняя адресация "какие именно простые" — ярлык.

**Ключевое наблюдение:** Плотность представлений растёт с n (42 = 5+37 = 11+31 = 13+29 = 19+23), что указывает на **естественную ψ-замкнутость**.

#### A1.2. Инвариант (ψ-покрытие)

Определим **φ-инвариант Гольдбаха**:
```
G_φ(n) := K_add(представления n как p₁+p₂) - K_mult(спектр простых ≤ n)
```

При **ψ-встраивании** простых в φ-решётку, аддитивная сшивка не требует внешних меток ⇒ G_φ ограничен константой.

#### A1.3. Лемма (ψ-сшивка простых)

Если простые числа реализуются как ψ-циклы на φ-графе (голономия), то их аддитивное комбинирование — внутренняя операция **без адресных ярлыков**. 

Отсутствие представления n = p₁ + p₂ при честной ψ-встройке потребовало бы **внешней метки** "несшиваемости" ⇒ ΔK > 0 ⇒ не min-K.

#### A1.4. Редукция (скелет)

1) Кодируем простые как φ-резонансы (связь с RH через ζ-функцию уже в Приложении B)
2) Аддитивная комбинация = ψ-суперпозиция двух резонансов
3) Для чётного n, отсутствие p₁+p₂ = n требует "блокировки" всех пар ⇒ внешний ярлык ⇒ ΔK > 0
4) При min-K блокировка невозможна ⇒ представление существует

#### A1.5. Фальсификация

- Найти чётное n > 2, где каждая попытка представления p₁+p₂ = n демонстрирует **снижение K** при **запрете** внешних ярлыков и TR=1.

#### A1.6. Мини-проверки

- Эмпирический скан: плотность представлений vs ψ-устойчивость кода
- Сравнение с "слабым Гольдбахом" (нечётные = 3 простых): тройная ψ-сшивка требует большего K

#### A1.7. Псевдокод аудита

```
Input: чётное n > 2
1) Построить φ-спектр простых p ≤ n
2) Для каждой пары (p, n-p): проверить ψ-сшиваемость без меток
3) Оценить G_φ(n)
4) fail, если нет представления И G_φ не растёт (ΔK = 0)
accept ⇔ существует p₁+p₂=n при min-K
```

### A2. Гипотеза о простых близнецах

**Классика:** Существует бесконечно много простых пар (p, p+2).

#### A2.1. Встраивание в T_all

**Сцена:** Простые близнецы — это **минимальные ψ-дуплеты** на φ-решётке простых. Разрыв в 2 — минимальный шаг, сохраняющий различимость.

#### A2.2. Инвариант (ψ-дуальность)

Определим **инвариант близнецовости**:
```
T_ψ(p, p+2) := K(пара как единица) - 2·K(одиночное p)
```

**min-K:** Если простые образуют φ-структуру, минимальное различие = 2 (наименьший шаг сохранения ψ-ориентации). Запрет дуплетов с разрывом 2 требует внешнего параметра "минимального расстояния" ⇒ ΔK > 0.

#### A2.3. Лемма (бесконечность дуплетов)

При ψ-замкнутости структуры простых, любое "конечное завершение" дуплетов потребует внешней границы ⇒ ярлык ⇒ ΔK > 0. Следовательно, при min-K дуплеты бесконечны.

#### A2.4. Редукция (скелет)

1) Простые = ψ-резонансы на φ-решётке
2) Дуплет (p, p+2) = минимальная ψ-пара
3) Конечность дуплетов ⇒ нужна "последняя пара" ⇒ внешний маркер ⇒ ΔK > 0
4) min-K запрещает конечность ⇒ дуплетов бесконечно много

#### A2.5. Фальсификация

- Доказать конечность простых близнецов **без** введения внешней границы и при **снижении** K относительно бесконечной модели.

### A3. Гипотеза Коллатца (3n+1)

**Классика:** Последовательность f(n) = n/2 (чётное), 3n+1 (нечётное) всегда достигает 1.

#### A3.1. Встраивание в T_all

**Сцена:** Операции f — это **ψ-переключение** между двумя режимами (чёт/нечёт). Память о пути — внутренняя голономия.

#### A3.2. Инвариант (ψ-сходимость)

Определим **Коллатц-потенциал**:
```
V_φ(n) := log₂(n) + α·(число нечётных шагов до сих пор)
```
где α ≈ log₂(3) - 1 ≈ 0.585.

**Ключ:** V_φ монотонно убывает при min-K (без внешних ярлыков). Цикл или расхождение потребовали бы внешней метки "особого n" ⇒ ΔK > 0.

#### A3.3. Лемма (ψ-голономия пути)

При честном ψ-встраивании (память о пути = голономия), каждый шаг f кодируется **внутри** без адресации предыдущих шагов. 

Расхождение или цикл требуют **внешней памяти** о специальной начальной точке ⇒ ΔK > 0 ⇒ не min-K.

#### A3.4. Редукция (скелет)

1) f = ABCD-цикл: A (×3+1), B (÷2), C (выбор режима), D (проекция)
2) Потенциал V_φ убывает при ψ-замкнутости
3) Достижение 1 — естественный min-K финал (нельзя дальше без новых правил)
4) Любое иное поведение требует внешних меток ⇒ ΔK > 0

#### A3.5. Фальсификация

- Найти n, где траектория расходится или зацикливается **без** роста K при ψ-встраивании и TR=1.

### A4. Kummer-Vandiver Conjecture

**Классика:** Для простого p, p не делит классовое число h⁺ = h⁺(Q(ζ_p)).

#### A4.1. Встраивание в T_all

**Сцена:** Классовое число — мера **неудачи уникальной факторизации** = мера внешних ярлыков в кольце. ψ-встраивание требует внутренней факторизации.

#### A4.2. Инвариант (φ-чистота)

```
K_class(p) := минимальная длина кода классов идеалов при ψ-замкнутости
```

**Тезис:** Делимость p | h⁺ означает, что структура классов содержит p-торсию ⇒ требует внешней "p-адресации" ⇒ ΔK > 0.

#### A4.3. Лемма

При min-K в кольце циклотомических целых, классовое число **не может** содержать p-кратную структуру без внешних меток. Следовательно, p ∤ h⁺.

#### A4.4. Редукция (скелет)

1) Классовое число h⁺ кодирует отклонение от UFD
2) p | h⁺ ⇔ существует элемент порядка p в группе классов
3) Это требует "p-циклическую метку" ⇒ ΔK > 0
4) min-K запрещает ⇒ p ∤ h⁺

### A5. Perfect Square Rational Distances Problem

**Классика:** Можно ли найти 4 точки на плоскости с попарными **целыми** расстояниями, образующими квадрат?

#### A5.1. Встраивание в T_all

**Сцена:** Рациональные расстояния = ψ-когерентные сшивки на φ-решётке. Квадрат = максимальная симметрия ⇒ минимум ярлыков.

#### A5.2. Инвариант (ψ-симметрия)

```
S_φ(конфигурация) := K(координаты) + K(расстояния) при ψ-замкнутости
```

Квадрат с целыми расстояниями требует **одновременной** ψ-когерентности по обоим осям ⇒ очень жёсткое условие.

#### A5.3. Лемма

Диагональ квадрата = √2 × сторона. Для целочисленных расстояний: d = √2 × a должно быть целым ⇒ √2 = d/a ⇒ рационально, что невозможно.

**φ-ψ интерпретация:** √2 — иррационален ⇒ требует бесконечного φ-развития ⇒ нельзя закодировать конечным K без внешней "обрезки" ⇒ ΔK > 0.

#### A5.4. Редукция

Целочисленный квадрат на плоскости требует рационализации √2, что при min-K без внешних ярлыков невозможно ⇒ **не существует**.

## РАЗДЕЛ B: ГЕОМЕТРИЯ

### B1. Union-Closed Sets Conjecture (Frankl)

**Классика:** В любом семействе множеств, замкнутом относительно объединения, существует элемент, принадлежащий ≥ половине множеств.

#### B1.1. Встраивание в T_all

**Сцена:** Замыкание относительно ∪ — это **ψ-аккумуляция** (накопление без потерь). Элемент высокой частоты — естественный центр масс.

#### B1.2. Инвариант (ψ-баланс)

```
B_φ(семейство) := max_x (частота x) / |семейство|
```

**Тезис:** При ψ-замкнутости (∪-замыкание), B_φ ≥ 1/2 без внешних меток.

#### B1.3. Лемма (центр масс)

Union-замкнутое семейство имеет **монотонную φ-структуру**: каждое объединение увеличивает покрытие. При min-K без внешних адресаций элементов, центр масс естественно возникает на частоте ≥ 1/2.

#### B1.4. Редукция

1) ∪-замкнутость = φ-накопление
2) Отсутствие элемента с частотой ≥ 1/2 требует **равномерного распределения**
3) Равномерность при φ-накоплении требует внешней "балансировки" ⇒ ΔK > 0
4) min-K ⇒ центр масс существует

### B2. Toeplitz' Conjecture (вписанный квадрат)

**Классика:** Любая замкнутая кривая на плоскости содержит 4 точки, образующие квадрат.

#### B2.1. Встраивание в T_all

**Сцена:** Замкнутая кривая = непрерывная ψ-голономия. Квадрат = максимальная симметрия = min-K конфигурация из 4 точек.

#### B2.2. Инвариант

```
Q_φ(кривая) := минимальная стоимость кодирования 4 точек с "квадратным" ограничением
```

При ψ-непрерывности, квадрат — естественная конфигурация min-K.

#### B2.3. Редукция

1) Замкнутая кривая = ψ-замкнутый путь на плоскости
2) Квадрат = конфигурация минимальной сложности среди 4-точечных симметрий
3) Отсутствие квадрата требует "блокировки всех квадратов" ⇒ внешние метки ⇒ ΔK > 0
4) min-K ⇒ квадрат существует

### B3. Kneser-Poulsen Conjecture (3D)

**Классика:** Сжатие конфигурации единичных шаров не увеличивает объём объединения.

#### B3.1. Встраивание в T_all

**Сцена:** Объём = мера **различимости** в пространстве. Сжатие = уменьшение ψ-разнообразия.

#### B3.2. Инвариант

```
V_φ(конфигурация) := объём объединения / (число шаров × объём единичного шара)
```

**Тезис:** При ψ-замкнутости и сжатии расстояний, V_φ не может расти без внешних меток.

#### B3.3. Лемма

Сжатие расстояний = уменьшение K-сложности конфигурации. Рост объёма при уменьшении K требует внешней "инфляции" ⇒ ΔK > 0 ⇒ не min-K.

#### B3.4. Редукция

1) Сжатие конфигурации ⇒ K ↓
2) Объём = функция K при ψ-замкнутости
3) V растёт при K ↓ ⇒ нарушение монотонности ⇒ требует внешних меток
4) min-K ⇒ V не растёт при сжатии

### B4. Perfect Cuboid Problem

**Классика:** Существует ли прямоугольный параллелепипед с целочисленными рёбрами, диагоналями граней и пространственной диагональю?

#### B4.1. Встраивание в T_all

**Сцена:** Целочисленность всех расстояний = ψ-когерентность в 3D. Требуется одновременная рационализация √2, √3, √6.

#### B4.2. Инвариант

Для параллелепипеда (a, b, c):
- Диагонали граней: √(a²+b²), √(b²+c²), √(a²+c²)
- Пространственная: √(a²+b²+c²)

Все должны быть целыми.

#### B4.3. Лемма (φ-несовместность)

Система уравнений:
```
a² + b² = d₁²
b² + c² = d₂²
a² + c² = d₃²
a² + b² + c² = D²
```

При φ-анализе: требует одновременной целочисленности корней с вложенными радикалами. Без внешних "подгоночных" констант K → ∞.

#### B4.4. Редукция

Целочисленный perfect cuboid требует бесконечного K для кодирования всех условий без внешних ярлыков ⇒ **не существует** при min-K.

### B5. Kissing Number Problem (высшие D)

**Классика:** Максимальное число единичных сфер, касающихся центральной единичной сферы в размерности D.

#### B5.1. Встраивание в T_all

**Сцена:** Kissing number = максимальная ψ-плотность при контакте = оптимальное φ-покрытие без перекрытий.

#### B5.2. Инвариант

```
κ_D := kissing number в размерности D
```

**Связь с φ:** При D → ∞, структура упаковки должна оптимизироваться через φ-симметрии.

#### B5.3. Лемма

Известно: κ₁=2, κ₂=6, κ₃=12, κ₈=240, κ₂₄=196,560.

φ-гипотеза: κ_D связано с φ-структурой через:
```
κ_D ~ φ^(αD) для некоторого α
```

#### B5.4. Редукция

1) Kissing number = задача оптимальной ψ-упаковки
2) При min-K упаковка следует φ-симметриям
3) В известных размерностях (8, 24) — специальные решётки (E₈, Leech) = φ-структуры
4) Общая формула требует φ-классификации решёток

## РАЗДЕЛ C: АНАЛИЗ

### C1. Invariant Subspace Problem

**Классика:** Имеет ли каждый ограниченный оператор T на комплексном банаховом пространстве нетривиальное инвариантное подпространство?

#### C1.1. Встраивание в T_all

**Сцена:** Инвариантное подпространство = **ψ-стабильный канал** под действием оператора. Отсутствие такового требует абсолютной "перемешивающей" динамики.

#### C1.2. Инвариант

```
I_ψ(T) := минимальная размерность подпространства V: T(V) ⊆ V
```

**Тезис:** При ψ-замкнутости (обратимость/память внутри), I_ψ > 0 всегда.

#### C1.3. Лемма

Оператор без инвариантных подпространств = **полное перемешивание** = ψ-хаос. Для кодирования такого оператора при min-K потребуется бесконечное описание (нет редукции через подструктуры) ⇒ ΔK → ∞.

#### C1.4. Редукция

1) Любой ограниченный T имеет конечное K-описание
2) min-K достигается через разложение на инвариантные части
3) Отсутствие инвариантных подпространств ⇒ K → ∞
4) Противоречие ⇒ инвариантное подпространство существует

### C2. HRT Conjecture (Heisenberg-Riesz-Turan)

**Классика:** Для ненулевой L²-функции и конечного набора точек time-frequency плоскости, по крайней мере одна точка имеет ненулевое значение STFT.

#### C2.1. Встраивание в T_all

**Сцена:** STFT = φ-вейвлет (time-frequency локализация). Нули на всех точках = "ψ-блокировка" информации.

#### C2.2. Инвариант

```
H_φ(f, {(t_k, ω_k)}) := минимальное K для кодирования f через STFT-значения в точках
```

**Тезис:** Все нули ⇒ H_φ → ∞ (невозможно восстановить f без внешних данных).

#### C2.3. Редукция

1) Ненулевая f имеет конечное K
2) STFT = φ-разложение
3) Все нули в конечном наборе точек ⇒ требуется бесконечно много других точек для восстановления
4) ΔK → ∞ ⇒ не min-K ⇒ хотя бы одна точка ненулевая

## РАЗДЕЛ D: АЛГЕБРА

### D1. Inverse Galois Problem

**Классика:** Является ли каждая конечная группа G группой Галуа расширения ℚ?

#### D1.1. Встраивание в T_all

**Сцена:** Группа Галуа = **ψ-симметрии** поля. Вопрос: можно ли все конечные симметрии реализовать над ℚ без внешних полей?

#### D1.2. Инвариант

```
K_Gal(G) := минимальная сложность конструкции расширения F/ℚ с Gal(F/ℚ) ≃ G
```

**Тезис:** При ψ-замкнутости (симметрии внутри), любая конечная группа реализуема.

#### D1.3. Лемма

Конечная группа G = конечный набор ψ-операций. Если ℚ + φ-структура достаточно богаты, любая конечная ψ-конфигурация реализуема без внешних полей (min-K).

#### D1.4. Редукция

1) G = конечная группа ⇒ конечное K
2) ℚ + алгебраическое замыкание = φ-полная структура
3) Нереализуемость G требует "дыры" в φ-структуре ⇒ внешнее ограничение ⇒ ΔK > 0

**Статус:** Частично решена для многих G; общий случай — открыт.

### D2. Bounded Burnside Problem

**Классика:** Для каких (m, n) свободная группа Бёрнсайда B(m,n) конечна?

#### D2.1. Встраивание в T_all

**Сцена:** B(m,n) = все m-порождённые группы с x^n = 1. Конечность = ψ-замкнутость без бесконечного роста.

#### D2.2. Инвариант

```
|B(m,n)| = размер группы (конечна или ∞?)
```

**φ-анализ:** При n нечётном и большом, возникают "дикие" подгруппы = бесконечные ψ-разветвления.

#### D2.3. Редукция

1) Малые n (2, 3, 4, 6): B(m,n) конечна (доказано)
2) n = 5: B(2,5) — открыт
3) Большие нечётные n: бесконечна (Новиков-Адян, Зельманов)

**φ-критерий:** B(m,n) конечна ⇔ ψ-структура замыкается при min-K.

## РАЗДЕЛ E: КОМБИНАТОРИКА И ЛОГИКА

### E1. 1/3–2/3 Conjecture

**Классика:** В любом конечном частично упорядоченном множестве (не полностью упорядоченном) существуют x, y: вероятность x < y в случайном линейном расширении ∈ [1/3, 2/3].

#### E1.1. Встраивание в T_all

**Сцена:** Частичный порядок = ψ-структура с неопределённостями. Линейные расширения = полные ψ-ориентации.

#### E1.2. Инвариант

```
P_φ(x, y) := вероятность x < y в случайном линейном расширении
```

**Тезис:** При ψ-замкнутости (все расширения равновероятны), P_φ не может быть экстремальной без внешних меток.

#### E1.3. Лемма

P(x < y) ∈ {0, 1} только если порядок между x и y **фиксирован**. Для несравнимых x, y без внешних "весов", распределение P_φ стремится к центральным значениям.

Экстремальные значения вблизи 0 или 1 требуют внешней "асимметрии" ⇒ ΔK > 0.

#### E1.4. Редукция

1) Неполный порядок ⇒ существуют несравнимые пары
2) Для несравнимой пары min-K ⇒ нет предпочтения ⇒ P ≈ 1/2
3) Диапазон [1/3, 2/3] = естественная флуктуация без внешних меток
4) Хотя бы одна пара в этом диапазоне

def psi_test_RH(rho, epsilon=1e-6):

    Проверка ψ-инвариантности нуля ρ

    xi_s = compute_xi(s)
